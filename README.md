📖 Overview

This project focuses on converting spoken language into Indian Sign Language (ISL) using machine learning techniques. It helps bridge the communication gap between hearing and speech-impaired individuals and others

🎯 Objectives

~Convert speech into text using Speech Recognition

~Translate text into ISL gestures

~Improve accessibility for deaf and mute individuals


~Provide real-time communication support

🧠 Working of the Project

Speech Input – User gives voice input

Speech-to-Text Conversion – Converts audio into text

Text Processing – Removes unnecessary words and formats sentence

ISL Conversion – Maps text to sign language gestures
